<Warning>
  `hyve use` has been removed. The replacement is `hyve cluster auth <name>`.
</Warning>

## Replacement

Use `hyve cluster auth <name>` to authenticate against a cluster. This runs the module's auth operation (for example, `civo kubernetes config --save` for Civo clusters) and merges the result directly into `~/.kube/config`.

```bash
hyve cluster auth my-cluster
```

## Related Commands

- [hyve cluster](/docs/cli/cluster) — full cluster management including `auth`
- [hyve kubeconfig](/docs/cli/kubeconfig) — `hyve kubeconfig remove <name>` to clean up `~/.kube/config`
